## 1. デザイン方針

### 1.1 ビジュアルコンセプト

**時代背景**: 2000年代初頭の個人ページ・同人サイト的なビジュアル

| 要素 | 方針 |
|------|------|
| 色調 | **暗黒系**: 黒背景に水色/青緑色のテキスト。MSゴシック等モノスペース字体。 |
| レイアウト | **2カラム**: 左サイドバー（ナビ+リンク+カウンター） + 右コンテンツ |
| フォント | MS Pゴシック, Courier New（等幅フォント）|
| 装飾 | テーブル枠線、シンプルな罫線 |
| バナー | Google リンクのみ（1個）、テキストリンク表示 |
| アクセスカウンター | 昭和的なカウンター（ローテーション数字表示） |

### 1.2 ページ構成の全体像

```
ホーム
├─ 掲示板（BBSスレッド一覧）
│  └─ 掲示板スレッド個別ページ
├─ 日記（日記一覧）
│  └─ 日記個別ページ
├─ 倉庫（認証者のみ、6アイテム）
├─ 管理画面（ログイン/ログアウト）
│  └─ 端末ハブ（Case選択）
│     └─ Case03～05（謎ページ）
└─ ログアウト（確認画面）
```

### 3.2 掲示板（BBS）

**パス**: `#/bbs`

**構造**:
```
【左サイドバー】        【スレッド一覧】
                       ┌──────────────────┐
│ 【 ナビ 】          │ 掲示板             │
│                    │ 全スレッド表示     │
│ ・ ホーム          │                  │
│ ・ 掲示板          │ [998] 雑談スレ     │
│ ・ 日記            │ [997] 攻略スレ     │
│ ・ 倉庫            │ [996] 何か怖い...  │
│ ・ 管理人           │ [995] 日常        │
│                    │                  │
│ [33333]           │ 【 新規スレッド 】 │
│ COUNTER            │ （認証ユーザのみ） │
│                    │ タイトル: ______  │
│                    │ [     送信    ]  │
└────────┘          └──────────────────┘
```

**スレッド一覧**:
- 各スレッドは [番号] タイトル 形式
- 番号でソート（新規順）
- クリックで「スレッド個別ページ」へ遷移

**新規スレッド作成**（認証ユーザのみ）:
- 認証状態でのみ「新規スレッド」フォーム表示
- タイトル入力後、新スレッド作成（投稿者名固定「管理人」）
- 作成後は自動的に新スレッドに遷移

### 3.6 倉庫（Warehouse）

**パス**: `#/warehouse`

**認証要件**: 認証ユーザのみアクセス可能（パスワード「ほろう」または「ホロウ」で認証）

**構造**:
```
【左サイドバー】        【倉庫アイテム】
                       ┌──────────────────┐
│ 【 ナビ 】          │ 倉庫               │
│                    │ ──────────────    │
│ ・ ホーム          │                  │
│ ・ 掲示板          │ 🔐       🔐       │
│ ・ 日記            │ ???     ???      │
│ ・ 倉庫            │                  │
│ ・ 管理人           │ 🔐       🔐       │
│                    │ ???     ???      │
│ [33333]           │                  │
│ COUNTER            │ 🔐       🔐       │
│                    │ ???     ???      │
│                    │                  │
└────────┘          └──────────────────┘
```

**アイテム仕様**:
- **非認証時**: 全6アイテムを🔐アイコン + "???" で表示（ロック状態）
- **認証時**: 実際の画像 + タイトル を表示（説明は非表示）

**含まれるアイテム** (6個):
1. 第一の鍵 (🗝️ アイコン)
2. 古い日誌 (📔 アイコン)
3. 謎の手紙 (✉️ アイコン)
4. 暗号表 (🔤 アイコン)
5. 写真 (📷 アイコン)
6. 音声記録 (🎙️ アイコン)

## 5. JavaScriptロジック

### 5.1 新規実装機能

#### カウンター表示

```javascript
// counter.js
const AccessCounter = {
  get() {
    return 33333;  // 固定値
  },
  
  increment() {
    return 33333;  // インクリメント無効（常に 33333）
  },
  
  getFormatted() {
    return this.get().toString().padStart(5, '0');  // "33333"
  }
}
```

#### BBS スレッド管理

// threads.js （新規作成）

## 8. デザイン参考

2000年代初頭の個人ページ的な特徴:
- **モノクロ/淡色**: 黒背景 + 青/水色の文字
- **2カラムレイアウト**: CSS Flexbox による左右2分割
